AULA 1

O que é Hypertext Markup Language? (HTML)?
Derivado do SGML implementa a relação de links entre páginas e diversas outras tags pra estruturação de telas.

Como e por que surgiu o HTML?
O HTML foi criado em 1991 por Tim Berners-Lee como parte de um projeto maior que incuía o desenvolvimento do primeiro navegador, propôs um hipertexto para facilitar a troca e o gerenciamento de informações entre os usuários.

O que é Extensible Hypertext Markup Language (XHTML?)
É uma reformulação do HTML usando XML, ele foi desenvolvido para combinar a flexibilidade do HTML com a rigidez e extensibilidade do XML.

Qual a diferença de XHTML para HTML?
A sintaxe e as regras de estruturação dos documentos.

O que é Extensible Markup Language (XML)?
É uma linguagem de marcação criada pelo W3C para armazenar e transportar dados de forma estruturada e independente de plataforma.

Qual a diferença de HTML para XML? A HTML é usada principalmente para desenvolver a interface de usuário de uma aplicação. Por outro lado, o objetivo principal da XML é o intercâmbio e a transferência de dados. Ela codifica dados em um formato legível por máquinas e humanos. A XML descreve o que são os dados, enquanto a HTML determina como exibir esses dados para o usuário final.
